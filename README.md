# Shark Explorer 

Shark Explorer is a React app that lets users search and explore different shark species. Users can view a list of sharks and click on one to see more details like diet, habitat, lifespan, and scientific name.

## Features

- Search for sharks
- View shark details
- Simple and clean UI
- Images for selected sharks

## Tech Used

- React
- React Router
- JavaScript
- API Ninjas Animals API

## Setup

1. Clone the repo
2. Run `npm install`
3. Run `npm start`

## Notes

Some sharks may not have images depending on available data from the API.
